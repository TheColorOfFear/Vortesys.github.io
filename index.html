Vortesys

Welcome to Vortesys.

Vortesys is an organization committed to developing and maintaining desktop software for Windows. While the primary projects at the moment are shells, an entire suite of programs is planned out to replace common desktop applications with superior and more performant alternatives, such as an MP3 player, calculator, notepad and a task manager.

The primary project under development at the moment is Program Manager II, a successor to the original Program Manager shell. It has been rewritten for stability, usability and customizability. It is currently in early alpha.

The secondary project, currently in early development, is Modular Desktop. Modular Desktop is designed around the idea of modularity through toolbars. Each element of the desktop is a toolbar, and the user interface is entirely up to the user, their desired layout, and their installed toolbars.
