Vortex Systems

Welcome to Vortex Systems.

Vortex Systems is an organization committed to developing and maintaining desktop software for Windows. While the primary project at the moment is a shell, an entire suite is planned out to replace common desktop applications with superior and more performant alternatives.

The primary project under development at the moment is Program Manager II, a successor to the original Program Manager shell. It has been rewritten for stability, usability and customizability. It is currently in early alpha.
